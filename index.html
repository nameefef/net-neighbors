CONCEPT

什么是网上邻居？

我们提供一种基于 IP 地址与 /24 网段 的用户发现与留言机制，帮助你 在不依赖复杂社交关系的前提下，找到"网络上离你最近的人"。

当你开启该功能后，同一 IP 或同一 /24 网段内的用户可以相互留言与互动， 从而发现潜在的关联关系。它就像一块 "网络公告板"， 让同一网络环境下的用户自然相遇、按需交流。

FAQ

常见问题

你可能想了解的一些事情。

我的 IP 地址会被暴露给其他人吗？
不会。你只会看到"同 /24 网段"这个概念，系统不会展示你的完整 IP 地址给其他用户。所有展示均经过掩码处理，确保隐私安全。
如果我的 IP 经常变化怎么办？
当你的 IP 发生变化时，系统会自动检测并切换到新的 /24 网段社区。你可以选择是否保留在原社区的留言记录。
同一个 /24 网段能有多少用户？
/24 网段理论上最多包含 254 个 IP 地址，但实际活跃用户数量取决于该网段的使用情况。运营商大出口 NAT 下可能有数百甚至上千用户，而家庭宽带通常只有少数几个人。
可以屏蔽某些用户吗？
可以。你可以屏蔽特定用户的消息，被屏蔽的用户不会知道。同时我们也设置了举报机制，对于恶意行为我们会进行社区封禁。
如何关闭这个功能？
你可以在设置中一键关闭"网上邻居"功能。关闭后，系统将停止检测你的 IP 网段，你的留言将变为匿名状态，其他用户无法再看到你的任何信息。
